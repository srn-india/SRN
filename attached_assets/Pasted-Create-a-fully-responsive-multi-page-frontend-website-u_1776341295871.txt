Create a fully responsive, multi-page frontend website using React.js with Vite (NO Next.js) and Tailwind CSS for a non-profit organization named "Sashakt Rashtra Nirman" (सशक्त राष्ट्र निर्माण). No backend, no database, no authentication. Clean component-based architecture with modern UI.

📁 Tech Stack

Framework: React + Vite
Styling: Tailwind CSS (configured properly in vite + tailwind config)
Routing: React Router v6
Icons: Lucide React
Animations: Framer Motion

अनुसंधान, प्रशिक्षण एवं नवाचार / Research, Training & Innovation

सहभागिता एवं सहयोग / Participation & Collaboration

Work with government bodies, NGOs, educational institutions, CSR units, experts
